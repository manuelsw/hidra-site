About me

Nato in provincia di La Spezia, mi sono avvicinato al mondo dell'informatica nel 2016 attraverso i file batch, creando script che facilitassero l'utilizzo del computer e che creassero collegamenti diretti con email, siti bancari ecc. Nel 2018 ho imparato Html e Css, integrandole poi l'anno successivo con il framework Bootstrap. Nel 2019 ho inoltre imparato javascript, che mi ha permesso di riprodurre Rock Paper Scissor e Guess the Number in una pagina html. Nel 2020 ho imparato python e le sue librerie, iniziando a creare GUI e bot discord. Con tkinter ho creato uno script per nascondere files tramite password e un'interfaccia che ti permettesse di spammare messaggi agli amici. Nel 2021 ho seguito un corso java ed ho imparato la creazione di plugin minecraft.
